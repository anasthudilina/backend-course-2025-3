Лабораторна робота №3: Backend Course 2025



Застосунок на Node.js для обробки даних про автомобілі з файлу JSON.



Як запустити

1\. Встановіть залежності: `npm install`

2\. Запустіть програму з обов'язковим параметром входу:

&#x20;  ```bash

&#x20;  node main.js -i mtcars.json -d

Параметри

\-i, --input — шлях до файлу (обов'язково)



\-o, --output — запис результату у файл



\-d, --display — вивід у консоль



\-c, --cylinders — показати кількість циліндрів



\-m, --mpg — фільтр за паливною економністю
